# Database Models

Cally uses **PostgreSQL 16** with **Prisma 7** and the `@prisma/adapter-pg` driver. All models use **cuid** IDs.

## Domain Overview

The 24 Prisma models are organized into five domains:

| Domain | Models | Purpose |
|---|---|---|
| Identity & Access | User, Account, Session, Verification, TwoFactor, Passkey | better-auth core tables |
| Multi-Tenancy | Organization, Member, Invitation, Team, TeamMember | Organization plugin |
| Calendar Core | Event, Availability, DateOverride, AppointmentType, Appointment, CalendarStream | Core scheduling |
| External Sync | CalendarAccount, ExternalCalendar, ExternalEvent, CanonicalEvent, ExternalEventMapping | Calendar integrations |
| AI Layer | UserAICredential, AIConversation, AIMessage | BYO-AI |
| Misc | AuditLog, Settings, ApiKey | Administration |

## Identity & Access

### User

The central user entity with many relations.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `email` | String | Unique, indexed |
| `name` | String | Display name |
| `slug` | String | Unique, URL-friendly identifier |
| `role` | String | `"ADMIN"` or `"USER"` |
| `timezone` | String | IANA timezone (e.g. `"America/New_York"`) |
| `theme` | String | `"light"`, `"dark"`, or `"system"` |
| `country` | String? | ISO country code |
| `avatarUrl` | String? | Profile image URL |
| `onboardingCompleted` | Boolean | Defaults to `false` |
| `twoFactorEnabled` | Boolean | Defaults to `false` |
| `failedLoginAttempts` | Int | For account lockout |
| `lockedUntil` | DateTime? | Account lockout expiration |
| `notificationPrefs` | Json? | Notification preferences |
| `icalToken` | String? | Token for iCal feed access |

**Relations:** Has many Account, Session, Event, Availability, DateOverride, AppointmentType, Appointment, CalendarAccount, CalendarStream, UserAICredential, AIConversation, ApiKey, Member, AuditLog.

### Account

OAuth/credential provider links. Unique on `(providerId, accountId)`.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Foreign key → User |
| `accountId` | String | Provider-specific user ID |
| `providerId` | String | e.g. `"google"`, `"github"`, `"credential"` |
| `accessToken` | String? | OAuth access token |
| `refreshToken` | String? | OAuth refresh token |
| `idToken` | String? | OpenID ID token |
| `password` | String? | Bcrypt hash for credential accounts |

### Session

Server-side session tokens with device tracking.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `token` | String | Unique session token |
| `userId` | String | Foreign key → User |
| `expiresAt` | DateTime | Session expiration |
| `ipAddress` | String? | Login IP |
| `userAgent` | String? | Browser/device info |

### Verification

Email verification and password reset tokens.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `identifier` | String | Email or identifier |
| `value` | String | Token/hash value |
| `expiresAt` | DateTime | Token expiration |

### TwoFactor

TOTP two-factor authentication secrets and backup codes (encrypted at rest).

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Unique, foreign key → User |
| `secret` | String | AES-256-GCM encrypted TOTP secret |
| `backupCodes` | String? | Encrypted backup codes |

### Passkey

WebAuthn passkey credentials.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Foreign key → User |
| `credentialId` | String | Unique WebAuthn credential ID |
| `publicKey` | Bytes | WebAuthn public key |
| `counter` | Int | Sign count for replay protection |
| `transports` | String? | Allowed transports |
| `backedUp` | Boolean | Whether the key is backed up |

### Event

| Field | Type | Notes |
|---|---|---|

Weekly availability slots that define when a user is bookable.

| Field | Type | Notes |
|---|---|---|

| `endTime` | String | HH:mm format |
| `isActive` | Boolean | Defaults to `true` |
| `userId` | String | Foreign key → User |

| Field | Type | Notes |

| `endTime` | String? | Override end time |

## External Calendar Sync

### CalendarAccount

OAuth-connected external calendar accounts.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Foreign key → User |
| `provider` | String | `"google"`, `"microsoft"`, `"apple_caldav"`, `"generic_caldav"` |
| `externalAccountId` | String | Provider-specific account ID |
| `email` | String | Account email |
| `accessTokenEnc` | String | AES-256-GCM encrypted access token |
| `refreshTokenEnc` | String | AES-256-GCM encrypted refresh token |
| `syncStatus` | String | `"idle"`, `"syncing"`, `"error"`, `"disconnected"` |
| `lastSyncAt` | DateTime? | Last successful sync |
| `webhookChannelId` | String? | Google push notification channel |
| `webhookExpiresAt` | DateTime? | Webhook channel expiration |

Unique on `(userId, provider, externalAccountId)`.

### ExternalCalendar

Individual calendars within a connected account.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `calendarAccountId` | String | Foreign key → CalendarAccount |
| `externalId` | String | Provider calendar ID |
| `summary` | String | Calendar display name |
| `timezone` | String? | Calendar timezone |
| `isPrimary` | Boolean | Whether this is the primary calendar |
| `selected` | Boolean | Whether the user has enabled this calendar for conflicts |
| `writable` | Boolean | Whether Cally can write events to this calendar |
| `syncToken` | String? | Incremental sync token |

Unique on `(calendarAccountId, externalId)`.

### ExternalEvent

Mirrored events from external calendars used for conflict detection.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `externalCalendarId` | String | Foreign key → ExternalCalendar |
| `externalId` | String | Provider event ID |
| `iCalUid` | String? | iCal UID for recurrence matching |
| `summary` | String? | Event title |
| `start` | DateTime | Start time |
| `end` | DateTime | End time |
| `allDay` | Boolean | Whether this is an all-day event |

Unique on `(externalCalendarId, externalId)`.

### CanonicalEvent

Deduplicated unified events across all providers for a user.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Foreign key → User |
| `title` | String? | Unified event title |
| `start` | DateTime | Start time |
| `end` | DateTime | End time |
| `allDay` | Boolean | All-day flag |
| `status` | String? | Event status |
| `iCalUid` | String? | iCal UID for cross-provider matching |

### ExternalEventMapping

Links canonical events to provider-specific events for deduplication.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `canonicalEventId` | String | Foreign key → CanonicalEvent |
| `provider` | String | Calendar provider |
| `externalCalendarId` | String | Foreign key → ExternalCalendar |
| `externalEventId` | String | Foreign key → ExternalEvent |
| `iCalUid` | String? | iCal UID |
| `rawHash` | String? | Hash of the raw provider data |

Unique on `(provider, externalCalendarId, externalEventId)`.

## AI Layer

### UserAICredential

Per-user encrypted AI provider API keys.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `userId` | String | Foreign key → User |
| `providerId` | String | `"openai"`, `"anthropic"`, `"google"`, `"groq"`, `"mistral"`, `"openrouter"`, `"ollama"` |
| `apiKeyEnc` | String | AES-256-GCM encrypted API key |
| `defaultModel` | String? | Default model ID for this provider |
| `baseUrl` | String? | Custom base URL (for Ollama/OpenRouter) |
| `label` | String? | User-friendly label |
| `lastTestedAt` | DateTime? | Last successful key validation |
| `lastTestedOk` | Boolean? | Whether the key was valid on last test |

Unique on `(userId, providerId, label)`.

| `pinned` | Boolean | Whether the conversation is pinned |

| `toolResults` | Json? | Tool result data |

Immutable audit trail for all significant actions.

| Field | Type | Notes |
|---|---|---|
| `id` | String (cuid) | Primary key |
| `action` | String | Action identifier |
| `entity` | String | Entity type |
| `entityId` | String | Entity ID |

| Field | Type | Notes |

| `name` | String | Key label |

| `keyPrefix` | String | First 8 chars for identification |
| `permissions` | Json? | Permission scope |
